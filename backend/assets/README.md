# PDF Assets Directory

This directory contains assets used for PDF generation.

## Subdirectories

- `fonts/` - Font files for PDF generation
- `images/` - Images (logos, icons) for PDF templates

## Image Guidelines

### Logos

Business logos should be:
- PNG format with transparency, or SVG for best quality
- Sized appropriately for receipt headers (recommended: 200-400px wide)
- Optimized for print (300 DPI equivalent)

### Adding Custom Assets

1. Place images in the appropriate subdirectory
2. Reference in templates using relative paths
3. Ensure images are optimized for size

## Recommended Image Assets

- `logo.png` - Business logo for receipts
- `logo-small.png` - Smaller logo variant for narrow receipts
