    Admin Panel Overview

    The Admin Panel is the central hub for managing your POS system. From here, administrators can configure products, manage users, view reports, and customize system settings.

    Accessing the Admin Panel

    The Admin Panel is only accessible to users with administrator privileges. To access the admin panel:

    1. Log in to the POS system with an administrator account
    2. Locate the Admin button in the top-right corner of the header
    3. Click the button to open the Admin Panel
    Permission Required: Only users with the Admin role can access the Admin Panel. Cashier accounts will not see the Admin button.

    Navigation Structure

    The Admin Panel features a sidebar navigation on the left side of the screen. Each section provides access to different management functions:

    Section Description Common Tasks
    Dashboard Overview of business performance View sales totals, check till status, manage business day
    Analytics Sales reports and trends View hourly sales charts, top products, category performance
    Products Product and variant management Add products, set prices, configure variants
    Categories Product category organization Create categories, set till visibility
    Stock Items Inventory item management Add ingredients, configure units, track quantities
    Inventory Stock level management Make adjustments, view stock levels, check history
    Users User account management Create users, assign roles, reset passwords
    Tills POS terminal management Create tills, assign devices, configure visibility

    Daily Closing End-of-day procedures Close business day, view closing history

    Switching Between Admin and POS Views

    Administrators often need to switch between the Admin Panel and the POS interface. The system provides easy navigation between these views:

    Returning to POS View

    1. Look for the Back to POS button in the header
    2. Click to return to the main sales interface
    3. Your order cart and session will be preserved

    Returning to Admin Panel

    1. From the POS view, click the Admin button
    2. You will be taken back to the Admin Panel
    Tip: You can quickly switch between views without losing your work. The system maintains your session state when navigating between Admin and POS views.

    Permission Requirements

    The Admin Panel enforces role-based access control. Different sections require different permission levels:

    Action Required Role
    View Admin Panel Admin
    View Dashboard/Analytics Admin
    Create/Edit/Delete Products Admin
    Create/Edit/Delete Users Admin
    Manage Stock/Inventory Admin
    Close Business Day Admin
    Modify Settings Admin
    Security Note: Admin accounts have full access to all system functions. Only grant admin privileges to trusted users who need complete system access.

    Data Refresh Behavior

    The Admin Panel displays real-time data in most sections. Understanding how data refreshes helps you work more effectively:

    • Dashboard: Sales totals update automatically as transactions occur
    • Product/Category Lists: Refresh when navigating to the section or after making changes
    • Transaction History: Shows transactions up to the current moment
    • Stock Levels: Update immediately when adjustments are made or products are sold

    To manually refresh data in any section, you can navigate away and back, or reload the page.
