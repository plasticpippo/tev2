    Inventory & Stock Management

    The Stock Items and Inventory sections allow you to manage your inventory. Track stock levels, make adjustments, and monitor consumption to maintain optimal inventory control.

    Overview

    The inventory system consists of two related sections:

    • Stock Items: Manage individual inventory items (ingredients and sellable goods)
    • Inventory: Make adjustments and track stock level changes

    Stock items are consumed automatically when products are sold (based on product variant configuration), and can be manually adjusted through the Inventory section.

    Stock Items Management

    Stock Items represent the individual inventory items in your establishment - ingredients used in products, supplies, and goods for resale.

    Creating Stock Items

    To create a new stock item:

    1. Navigate to Stock Items in the Admin Panel
    2. Click Add Stock Item
    3. Fill in the required information:
      • Name: The item name (e.g., "Coffee Beans", "Large Cups")
      • Type: Choose "Ingredient" or "Sellable Good"
      • Base Unit: The primary unit of measurement (e.g., "kg", "liters", "units")
      • Initial Quantity: Starting stock level
    4. Configure optional Purchasing Units if needed
    5. Click Save

    Units Configuration

    Stock items use base units for internal tracking and can have purchasing units for ordering.

    Base Units

    The base unit is the standard measurement used for this item. Common base units include:

    • kg - Kilograms (for solids like flour, sugar)
    • liters - Liters (for liquids like milk, oil)
    • units - Individual items (for cups, napkins)
    • grams - Grams (for precise measurements)
    • ml - Milliliters (for small liquid quantities)

    Purchasing Units

    Purchasing units define how the item is typically ordered. For example:

    • Base unit: grams, Purchasing: 1kg bag
    • Base unit: ml, Purchasing: 1L carton
    • Base unit: units, Purchasing: 50-pack

    Stock Level Indicators

    The system displays stock levels with visual indicators to help identify items needing attention:

    Indicator Meaning Action Required
    Green Stock level is healthy No action needed
    Yellow/Orange Stock is getting low Consider reordering soon
    Red Stock is critically low or out Reorder immediately
    Note: Stock level thresholds may be configurable in future versions. Currently, items are highlighted when quantity is low.

    Making Stock Adjustments

    Stock adjustments allow you to manually increase or decrease stock levels. This is useful for:

    • Recording deliveries
    • Correcting inventory counts
    • Writing off expired or damaged items
    • Initial stock setup

    Creating a Stock Adjustment

    1. Navigate to Inventory in the Admin Panel
    2. Click Add Adjustment
    3. Select the Stock Item
    4. Enter the Quantity (positive to add, negative to subtract)
    5. Provide a Reason for the adjustment
    6. Click Save

    Adjustment Reasons

    Common reasons for stock adjustments include:

    • Delivery Received: New stock from supplier
    • Inventory Count: Correction after physical count
    • Damaged/Expired: Items written off
    • Waste: Spoilage or preparation waste
    • Transfer: Moving stock between locations
    • Initial Setup: Setting initial stock levels

    Viewing Adjustment History

    The Inventory section maintains a complete history of all stock adjustments.

    Accessing History

    1. Navigate to Inventory
    2. View the adjustment history list
    3. Each entry shows:
      • Date and time of adjustment
      • Item name
      • Quantity change
      • Reason
      • User who made the adjustment

    Consumption Reports

    The Itemised Consumption section provides detailed reports on which stock items have been consumed over a period.

    Accessing Consumption Reports

    1. Navigate to Itemised Consumption in the Admin Panel
    2. Select a Date Range
    3. Optionally filter by:
      • Category: Filter by product category
      • Stock Item Type: Ingredients or Sellable Goods
    4. View the consumption data

    Understanding Consumption Data

    The report shows:

    • Total Consumption: How much of each stock item was used
    • By Product: Which products consumed the most
    • By Period: Daily, weekly, or custom date ranges

    Automatic Stock Consumption

    When products are sold, stock is automatically consumed based on the product variant configuration.

    How It Works

    1. Each product variant has Stock Consumption settings
    2. When an item is sold, the system looks up the consumption for that variant
    3. The specified quantity is automatically deducted from the stock item
    4. Stock levels update in real-time

    Example

    If a "Large Coffee" variant is configured to consume:

    • 20g Coffee Beans
    • 200ml Milk
    • 1 Large Cup

    Every time a Large Coffee is sold, these items are automatically deducted from inventory.

    Best Practices

    Stock Item Management

    • Create stock items for all items you want to track
    • Use consistent naming conventions
    • Set appropriate base units for accurate tracking
    • Configure purchasing units to match supplier packaging

    Stock Adjustments

    • Always provide a reason for adjustments
    • Regularly compare actual stock to system stock
    • Investigate large discrepancies
    • Document all adjustments for accountability

    Consumption Monitoring

    • Review consumption reports regularly
    • Compare actual usage to expected usage
    • Identify items with unusual consumption patterns
    • Use data to optimize ordering and reduce waste
    Tip: Regular inventory counts help ensure system accuracy. Perform physical counts periodically and compare to system quantities.
