    Till Management

    Tills represent physical POS terminals or stations in your establishment. This guide explains how to create and manage tills, assign devices, and configure visibility settings.

    Overview

    A "till" represents a physical POS terminal or station in your establishment. Each till:

    • Has its own product layout configuration
    • Can be assigned to specific devices or users
    • Can have different category visibility settings
    • Appears in the dashboard for status monitoring

    Viewing All Tills

    Navigate to the Tills section from the Admin Panel sidebar to see all configured tills.

    Till List Information

    Each till entry shows:

    • Till Name: The identifier for this terminal
    • Created: When the till was created
    • Actions: Edit and delete buttons

    Creating Tills

    To create a new till:

    1. Click the Add Till button
    2. Enter a Till Name (e.g., "Bar POS", "Counter 1", "Patio Terminal")
    3. Click Save

    Assigning Devices to Tills

    Each till can be associated with a specific device or workstation. This allows the system to identify which terminal is being used.

    Understanding Device-Till Relationship

    • Device Identification: The system identifies devices by their browser/device characteristics
    • Persistent Assignment: Once assigned, the device will automatically connect to the till on future logins
    • Multiple Devices: A single device can be reassigned to different tills as needed

    Assigning a Device

    1. Log in to the POS on the target device
    2. The system will prompt for till selection if not yet assigned
    3. Select the appropriate till from the list
    4. The device is now associated with that till

    Reassigning a Device

    If you need to change which till a device connects to:

    1. Log in as an administrator
    2. Navigate to the Tills section
    3. Edit the till and clear the device assignment
    4. The next login on that device will prompt for till selection

    Till Visibility Settings

    Till visibility is controlled through category settings. Each category can specify which tills can see products in that category.

    Configuring Category Visibility

    1. Go to the Categories section in Admin Panel
    2. Edit a category
    3. Find the Visible Tills setting
    4. Select which tills should display this category
    5. Save the category

    Visibility Behavior

    • All Tills: If no till is selected, the category is visible to all tills
    • Specific Tills: Only selected tills will show this category
    • Real-time Updates: Changes take effect immediately

    Editing Tills

    To modify a till:

    1. Locate the till in the tills list
    2. Click the Edit button
    3. Modify the till name
    4. Click Save
    Note: Changing the till name does not affect device assignments or existing data.

    Deleting Tills

    To delete a till:

    1. Locate the till in the tills list
    2. Click the Delete button
    3. Confirm the deletion
    Important: When you delete a till:
    • The till is removed from all category visibility settings
    • Any device assignments are cleared
    • Historical transaction data remains intact

    Monitoring Till Status

    You can monitor till status from the Dashboard.

    Till Status Indicators

    Status Description
    Active Till is currently processing transactions
    Idle Till is logged in but not actively processing
    Offline Till is not connected or not configured

    Best Practices

    Till Configuration

    • Create a till for each physical POS terminal
    • Use descriptive names that indicate location (e.g., "Bar 1", "Patio 1")
    • Configure category visibility based on what each terminal needs to sell

    Device Management

    • Assign devices to tills during initial setup
    • Document which devices are assigned to which tills
    • Reassign devices when hardware is moved or replaced

    Category Visibility

    • Configure category visibility to match each terminal's product offerings
    • For example, bar terminals might only show drink categories
    • Test visibility by logging into each till after configuration
    Tip: Regularly review till status from the Dashboard to ensure all terminals are functioning properly.
