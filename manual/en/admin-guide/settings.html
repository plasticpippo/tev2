    Tax Settings

    Configure how tax is calculated in your POS system. The tax setting affects all price calculations.

    Tax Modes

    Mode Description Example
    Tax Inclusive Prices entered include tax. The system calculates the base price. $10.00 price = $8.33 base + $1.67 tax
    Tax Exclusive Tax is added to entered prices. $10.00 base + $2.00 tax = $12.00 total
    No Tax Tax is disabled. Prices are final with no additional tax. $10.00 = $10.00

    Configuring Tax Mode

    1. Navigate to Settings in the Admin Panel
    2. Find the Tax section
    3. Select the desired tax mode from the dropdown
    4. Click Save

    Business Day Settings

    Configure how business days are managed in the system.

    Auto Start Time

    The auto start time determines when a new business day begins automatically:

    • At the specified time, any new transaction starts a new business day
    • Previous day's totals are finalized
    • Default: 6:00 AM

    Manual Closing

    Manual closing requires an administrator to explicitly close the business day:

    • When enabled, days must be closed manually
    • Provides more control over day boundaries
    • Useful for businesses with specific reporting needs

    Configuring Business Day

    1. Navigate to Settings in the Admin Panel
    2. Find the Business Day section
    3. Set the Auto Start Time (HH:MM format)
    4. Enable or disable Manual Closing
    5. Click Save

    Best Practices

    Tax Configuration

    • Set tax mode during initial setup
    • Test with sample transactions after changing
    • Ensure staff understand how prices are displayed
    • Consult with an accountant for tax compliance

    Business Day Timing

    • Set auto-start time to match your earliest opening
    • Consider reporting periods when choosing times
    • Ensure enough time between last transaction and next auto-start
    • Communicate schedule to all staff
    Tip: Test any changes to settings during a quiet period to ensure everything works as expected.
