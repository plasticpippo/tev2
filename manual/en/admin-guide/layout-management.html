    Layout Management

    Layout management allows you to customize how products appear on each till's POS screen. Create custom arrangements, share layouts between tills, and optimize the product grid for efficient service.

    Overview

    Layouts determine the visual arrangement of products in the POS interface. Each till can have its own layout, or you can create shared layouts that can be applied to multiple tills.

    Layout Types

    Type Description Use Case
    Till-Specific Layout Custom layout for a single till Unique configurations per terminal
    Shared Layout Reusable layout that can be applied to multiple tills Standard layouts across similar terminals

    Creating Till-Specific Layouts

    Each till can have its own custom layout for each category. The layout determines where each product variant appears in the grid.

    Accessing Layout Editor

    1. Log in to the POS on the target till
    2. Enter Admin Mode
    3. Navigate to the category you want to arrange
    4. Long-press or right-click on the product grid
    5. Select Edit Layout or use the layout editing interface

    Arranging Products

    1. Drag products to reposition them in the grid
    2. Products snap to a 4-column grid
    3. Multiple rows can be used
    4. Layout saves automatically

    Grid Configuration

    • Columns: Fixed at 4 columns
    • Rows: Unlimited - scroll as needed
    • Position: Specified by gridColumn (1-4) and gridRow (starting at 1)

    Creating Shared Layouts

    Shared layouts allow you to create a standard arrangement once and apply it to multiple tills.

    Creating a Shared Layout

    1. Navigate to Layout Management in the Admin Panel
    2. Click Create Shared Layout
    3. Enter a Name for the layout
    4. Select the Category this layout applies to
    5. Add products and specify their positions
    6. Click Save

    Shared Layout Options

    • Name: Descriptive identifier for the layout
    • Category: Which product category this layout arranges
    • Positions: Product-variant-to-grid-position mappings
    • Owner: User who created the layout

    Applying Shared Layouts

    Once you've created a shared layout, you can apply it to any till.

    Loading a Layout to a Till

    1. Navigate to Shared Layouts in the Admin Panel
    2. Find the layout you want to apply
    3. Click Load to Till or use the "Load to Till" action
    4. Select the target till from the list
    5. Confirm the action
    Note: Loading a shared layout replaces any existing layout for that till and category combination. The layout is copied, not linked, so changes to the shared layout won't affect tills where it's already loaded.

    Editing Layouts

    You can modify existing layouts at any time.

    Editing Till-Specific Layouts

    1. Log in to the POS on the target till
    2. Navigate to the category
    3. Enter layout edit mode
    4. Drag products to new positions
    5. Changes save automatically

    Editing Shared Layouts

    1. Navigate to Shared Layouts in Admin Panel
    2. Click Edit on the layout
    3. Modify the layout configuration
    4. Click Save

    Deleting Layouts

    Remove layouts when they're no longer needed.

    Deleting a Till Layout

    1. Log in to the POS on the target till
    2. Navigate to the category
    3. Enter layout edit mode
    4. Select Reset Layout or Clear Layout
    5. Confirm the action

    Deleting a Shared Layout

    1. Navigate to Shared Layouts in Admin Panel
    2. Click Delete on the layout
    3. Confirm the deletion
    Warning: Deleting a shared layout does not affect tills where it's already loaded. Those tills will retain their copy of the layout.

    Favourites Category

    The Favourites category is a special pseudo-category that shows all products marked as favourites, regardless of which regular category they belong to.

    Creating Favourites Layout

    1. Products must be marked as favourites in their variant settings
    2. The Favourites category (ID: -1) appears in the POS category list
    3. Create a layout for the Favourites category just like any other

    Layout for Favourites

    • All favourite products appear in this category
    • You can customize their arrangement
    • Useful for quick access to popular items

    Best Practices

    Layout Design

    • Place frequently ordered items in the top rows
    • Group related products together
    • Consider the reach zone for touch-screen terminals
    • Use consistent positioning across similar tills

    Shared Layouts

    • Create shared layouts for standard terminal configurations
    • Use different layouts for different terminal types (e.g., bar vs. counter)
    • Document which tills use which layouts
    • Update shared layouts when products change

    Maintenance

    • Review layouts when adding new products
    • Remove layouts for discontinued products
    • Test layouts by logging into each till
    • Consider staff feedback on layout efficiency
    Tip: Use shared layouts to maintain consistency across similar terminals, then make small adjustments for specific locations as needed.

    Troubleshooting

    Products Not Appearing

    If products aren't appearing in the POS grid:

    • Check that the category is visible for that till
    • Verify the product has at least one variant
    • Ensure the product is in an active category

    Layout Not Saving

    If layout changes aren't saving:

    • Check network connectivity
    • Ensure you're logged in with proper permissions
    • Try refreshing the page

    Shared Layout Not Applying

    If a shared layout isn't applying to a till:

    • Verify you're loading to the correct till
    • Check that the layout category matches the POS category
    • The till may have a specific layout overriding the shared one
