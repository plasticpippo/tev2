    Table & Room Management

    The Table & Room Management section allows you to create and organize your restaurant's physical layout. Define rooms, add tables, and arrange them visually to match your floor plan.

    Overview

    The Tables section in the Admin Panel provides three tabs for managing your dining area:

    • Rooms Tab: Create and manage dining rooms or areas
    • Tables Tab: Add and configure individual tables
    • Layout Tab: Visually arrange tables within rooms

    Managing Rooms

    Rooms represent distinct dining areas in your establishment. Examples include "Main Dining," "Patio," "Bar Area," or "Private Room."

    Viewing Rooms

    The Rooms tab displays all configured rooms with their descriptions and the number of tables assigned to each.

    Adding a New Room

    1. Go to the Rooms tab
    2. Click Add Room
    3. Enter a Room Name (required)
    4. Optionally add a Description for additional details
    5. Click Save
    Room Names: Room names must be unique. Choose descriptive names that staff can easily identify.

    Editing a Room

    1. Find the room in the list
    2. Click the Edit button
    3. Modify the name or description
    4. Click Save

    Deleting a Room

    1. Find the room in the list
    2. Click the Delete button
    3. Confirm the deletion
    Cannot Delete: You cannot delete a room that has tables assigned to it. First move or delete all tables in the room, then delete the room.

    Managing Tables

    Tables are individual seating locations within rooms. Each table can be assigned to orders and tracked for status.

    Viewing Tables

    The Tables tab shows all tables with their assigned room, status, and capacity.

    Adding a New Table

    1. Go to the Tables tab
    2. Click Add Table
    3. Enter the required information:
      • Table Name/Number: Identifier for the table (e.g., "Table 1", "T5", "Booth A")
      • Room: Select which room the table belongs to
    4. Configure optional settings:
      • Capacity: Number of seats at the table
      • Status: Initial status (usually "Available")
      • Width/Height: Visual size in the layout editor
    5. Click Save

    Table Status Options

    Status Description Visual Indicator
    Available Table is free and ready for guests Green
    Occupied Table has guests currently seated Red/Orange
    Reserved Table is reserved for upcoming guests Blue
    Unavailable Table is out of service (cleaning, maintenance) Gray

    Editing a Table

    1. Find the table in the list
    2. Click the Edit button
    3. Modify any of the table properties
    4. Click Save

    Deleting a Table

    1. Find the table in the list
    2. Click the Delete button
    3. Confirm the deletion
    Cannot Delete: You cannot delete a table that has an open tab associated with it. Close or transfer the tab first.

    The Layout tab provides a visual editor for arranging tables within rooms. This helps staff quickly locate tables and understand the restaurant layout.

    Layout Editor Modes

    The layout editor has three modes:

    Mode Function
    View Mode Preview the layout without making changes
    Edit Mode Select and modify table properties
    Drag Mode Drag tables to reposition them on the canvas

    Arranging Tables

    1. Select the Room you want to edit from the dropdown
    2. Switch to Drag Mode
    3. Click and drag tables to their desired positions
    4. Tables will snap to a grid for alignment
    5. Positions are saved automatically

    Resizing Tables

    1. Switch to Edit Mode
    2. Click on a table to select it
    3. Drag the corners or edges to resize
    4. Changes are saved automatically

    Keyboard Shortcuts

    Shortcut Action
    Ctrl+R Refresh data
    Ctrl+L Switch to Layout tab
    Ctrl+M Switch to Rooms tab
    Ctrl+T Switch to Tables tab
    Ctrl++ Add new table/room
    Ctrl+D Toggle layout mode

    Table Capacity

    Setting table capacity helps with reservation management and seating optimization.

    Setting Capacity

    1. Edit the table
    2. Enter the Capacity (number of seats)
    3. Save the changes

    Using Capacity Information

    • Helps staff assign appropriate tables to party sizes
    • Prevents overbooking reservations
    • Useful for planning seating arrangements

    Table Assignment in POS

    Once tables are configured, they can be used in the POS interface for order assignment.

    Assigning Orders to Tables

    1. In the POS view, add items to the order
    2. Click the Table button
    3. Select the room and table from the list
    4. The order is now associated with that table

    Table Status Updates

    Table status can change automatically based on activity:

    • When an order is assigned, status changes to Occupied
    • When the bill is paid, status returns to Available
    • Staff can manually change status as needed

    Best Practices

    Room Organization

    • Create rooms that match your physical layout
    • Use descriptive names that staff and customers understand
    • Consider service zones when defining rooms

    Table Naming

    • Use a consistent naming convention (e.g., "T1", "T2" or "Table 1", "Table 2")
    • Consider numbering tables in a logical sequence based on location
    • For special tables, use descriptive names (e.g., "Patio 1", "Bar Seat 1")

    Layout Design

    • Position tables to match the physical floor plan
    • Leave adequate space between tables in the layout
    • Size tables proportionally to their real-world size
    • Update the layout when physical changes are made
    Tip: A well-designed layout helps new staff learn the floor plan faster and reduces confusion during busy service.
