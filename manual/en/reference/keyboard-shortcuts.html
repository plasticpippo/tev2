    Keyboard Shortcuts

    Quick reference for keyboard shortcuts available throughout the POS system. Using keyboard shortcuts can significantly speed up your workflow and improve efficiency.

    Table Management Shortcuts

    These shortcuts are available in the Table & Room Management section of the Admin Panel. They help administrators quickly navigate between tabs and perform common actions.

    Shortcut Action Description
    Ctrl + R Refresh Data Reloads all table and room data from the server. Use this when changes made on another device are not visible.
    Ctrl + L Switch to Layout Tab Opens the visual layout editor where you can drag and position tables within rooms.
    Ctrl + M Switch to Rooms Tab Opens the rooms management section for creating and editing rooms.
    Ctrl + T Switch to Tables Tab Opens the tables management section for creating and editing individual tables.
    Ctrl + + Add New Item Creates a new table when on the Tables tab, or a new room when on the Rooms tab.
    Ctrl + D Toggle Layout Mode Cycles through layout modes: View (read-only), Edit (modify properties), and Drag (reposition tables).

    Product Grid Layout Shortcuts

    These shortcuts are available when editing Product Grid Layouts. They help with undo/redo operations and zooming the layout view.

    Shortcut Action Description
    Ctrl + Z Undo Reverts the last change made to the grid layout. Supports up to 50 levels of undo history.
    Ctrl + Shift + Z Redo Reapplies a change that was undone. Use after Ctrl+Z to restore a change.
    Ctrl + Y Redo (Alternative) Alternative shortcut for redo. Works the same as Ctrl+Shift+Z.
    Ctrl + + or Ctrl + = Zoom In Increases the zoom level of the layout view. Maximum zoom is 200%.
    Ctrl + - Zoom Out Decreases the zoom level of the layout view. Minimum zoom is 50%.
    Ctrl + 0 Reset Zoom Resets the zoom level back to 100% (default view).

    Moving Selected Items

    When a grid item is selected, use arrow keys to reposition it:

    Shortcut Action
    Arrow Keys Move selected item by 1 grid position in the direction pressed.
    Shift + Arrow Keys Move selected item by 2 grid positions for faster repositioning.

    General Shortcuts

    These shortcuts work throughout the application in various contexts.

    Shortcut Action Context
    Enter Confirm/Submit Submits forms, confirms dialog selections, or activates focused buttons.
    Escape Cancel/Close Closes modal dialogs, cancels ongoing operations, or deselects items.
    Space Select/Toggle Selects or toggles checkboxes, radio buttons, and other toggleable elements when focused.
    Tab Next Field Moves focus to the next interactive element in forms and dialogs.
    Shift + Tab Previous Field Moves focus to the previous interactive element in forms and dialogs.

    Tips for Using Shortcuts

    Best Practices
    • Learn the essentials first: Start with Ctrl+R (refresh) and Ctrl+D (toggle mode) for table management.
    • Use undo frequently: Ctrl+Z is your safety net when editing layouts. Experiment confidently knowing you can undo.
    • Keyboard navigation: Use Tab and Shift+Tab to navigate forms without reaching for the mouse.
    • Touch devices: On touchscreen devices, keyboard shortcuts may require an external keyboard. The virtual keyboard does not support shortcuts.
