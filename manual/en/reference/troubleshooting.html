    Troubleshooting

    Solutions to common issues you may encounter with the POS system. If you cannot resolve an issue using this guide, contact your administrator or technical support.

    Login Issues

    "Invalid username or password"

    Cause: The credentials entered do not match any user account.

    Solutions:

    • Check that Caps Lock is not enabled on your keyboard
    • Verify the username is spelled correctly (usernames are case-sensitive)
    • Re-enter your password carefully, watching for typos
    • Contact your administrator to verify your account exists and is active
    "Till not configured" or "Device not assigned"

    Cause: The device you're using hasn't been assigned to a till.

    Solutions:

    • Contact your administrator to assign this device to a till
    • The administrator can do this in Till Management
    • If you're an admin, log in from an already-assigned device to configure this one
    "User account is inactive"

    Cause: Your account has been deactivated by an administrator.

    Solutions:

    • Contact your administrator to reactivate your account
    • Administrators can change account status in User Management
    "Session has expired"

    Cause: Your login session has timed out due to inactivity or security policy.

    Solutions:

    • Simply log in again with your credentials
    • Your current order is automatically saved and will be restored
    • If this happens frequently, ask your administrator about session timeout settings
    "Token has been revoked. Please login again."

    Cause: Your session was invalidated, possibly because you logged in from another device or an administrator revoked your tokens.

    Solutions:

    • Log in again to create a new session
    • If you didn't log in elsewhere, contact your administrator for security review

    Payment Processing Errors

    "Payment processing failed"

    Cause: The transaction could not be completed due to a server or network issue.

    Solutions:

    • Check your network connection
    • Verify the transaction amount is correct
    • Try the transaction again
    • If using card payment, verify the card terminal is working
    • Contact your administrator if the problem persists
    "Invalid transaction data"

    Cause: The order contains invalid or incomplete information.

    Solutions:

    • Clear the order and rebuild it from scratch
    • Ensure all items have valid prices
    • Check that the order total is greater than zero

    Network Connectivity Problems

    Application not loading

    Cause: The server is unreachable or not responding.

    Solutions:

    • Check that your device is connected to the network
    • Try accessing other websites to verify internet connectivity
    • Refresh the page (press F5 or Ctrl+R)
    • Clear your browser cache and reload
    • Contact your IT support to verify the server is running
    "Database connection error"

    Cause: The server cannot connect to the database.

    Solutions:

    • This is a server-side issue - contact your administrator immediately
    • Do not attempt to process orders until the issue is resolved
    • Any open tabs will be preserved in the database
    "Service unavailable"

    Cause: The server is temporarily unable to handle requests.

    Solutions:

    • Wait a few moments and try again
    • The server may be restarting or under heavy load
    • Contact your administrator if the issue persists

    Data and Sync Issues

    Changes not appearing on other devices

    Cause: Data hasn't synchronized across devices yet.

    Solutions:

    • Refresh the page on the device showing outdated data (Ctrl+R)
    • Log out and log back in to force a full data refresh
    • Check network connectivity on both devices
    • Most data syncs within seconds, but may take longer on slow networks
    "Record not found" or "Resource not found"

    Cause: The item you're trying to access has been deleted or doesn't exist.

    Solutions:

    • Refresh the page to update your data
    • The item may have been deleted by another user
    • Return to the main list and select a different item
    "This value already exists" or "Duplicate" error

    Cause: You're trying to create an item with a name that already exists.

    Solutions:

    • Choose a different name for the item
    • Check if the item already exists in the list
    • Names must be unique within their category (products, categories, rooms, etc.)

    Permission and Access Issues

    "Access denied" or "Unauthorized"

    Cause: You don't have permission to perform this action.

    Solutions:

    • Verify you're logged in with the correct account
    • Some features require Admin role - check with your administrator
    • If you recently changed roles, log out and log back in
    "Admin privileges required"

    Cause: The action you're trying to perform requires administrator access.

    Solutions:

    • Contact an administrator to perform this action
    • If you are an admin, verify your account has the Admin role
    • Log out and log back in to refresh your permissions
    "Access denied. You do not own this table/layout."

    Cause: You're trying to access a resource assigned to another till.

    Solutions:

    • Each till has its own layouts and table assignments
    • Switch to the correct till or device
    • Contact your administrator if you need access to this resource

    Validation Errors

    "This field is required"

    Cause: A mandatory field was left empty.

    Solutions:

    • Fill in all required fields (marked with asterisks or highlighted)
    • Check for empty spaces that might look like empty fields
    "Invalid format" or "Invalid value"

    Cause: The entered data doesn't match the expected format.

    Solutions:

    • For numeric fields, enter only numbers (no letters or special characters)
    • For prices, use decimal format (e.g., 10.99)
    • For dates, use the date picker or the expected date format
    "Value is out of allowed range"

    Cause: The entered value is too large or too small.

    Solutions:

    • Check the minimum and maximum allowed values
    • For quantities, ensure the value is positive
    • For prices, ensure the value is not negative

    Rate Limiting

    "Too many requests" or "Too many write operations"

    Cause: You've made too many requests in a short time period. The system limits request rates to prevent abuse and ensure stability.

    Solutions:

    • Wait a few seconds before trying again
    • Avoid rapid repeated clicking or tapping
    • If using automated tools, reduce the request frequency
    • Contact your administrator if you consistently hit rate limits during normal use

    Deletion Issues

    "Cannot delete category with assigned products"

    Cause: The category still has products assigned to it.

    Solutions:

    • Reassign products to a different category first
    • Or delete the products if they're no longer needed
    • Once the category is empty, you can delete it
    "Cannot delete table with open tabs"

    Cause: The table has open tabs associated with it.

    Solutions:

    • Close or settle all open tabs for this table
    • Or reassign tabs to a different table
    • Once all tabs are closed, you can delete the table
    "Cannot delete room with assigned tables"

    Cause: The room still contains tables.

    Solutions:

    • Delete or reassign all tables in this room first
    • Once the room is empty, you can delete it
    "Cannot delete stock item. It is currently used in a product recipe."

    Cause: The stock item is linked to product variants for stock consumption tracking.

    Solutions:

    • Remove the stock item from all product variants that use it
    • Go to Product Management and edit each affected product
    • Once unlinked from all products, you can delete the stock item
    "Cannot delete your own account"

    Cause: You're trying to delete the account you're currently logged in with.

    Solutions:

    • Ask another administrator to delete your account
    • Or create a new admin account first, then delete this one

    When to Contact Support

    If you encounter any of the following situations, contact your administrator or technical support immediately:

    • Database errors: Any message mentioning "database connection error" or "database query error"
    • Internal server errors: Messages saying "Internal server error" or "An unexpected error occurred"
    • Security concerns: If you suspect unauthorized access or see unexpected login activity
    • Data loss: If transactions, tabs, or other data appear to be missing
    • Persistent issues: Problems that continue after following the troubleshooting steps above

    When contacting support, provide:

    • The exact error message you saw
    • What you were trying to do when the error occurred
    • The time the issue started
    • Your device and browser information
