    Quick Start Guide

    Get up and running with the POS system quickly. This guide walks you through the essential steps to start taking orders and processing payments in minutes.

    Quick Start Checklist

    Follow this checklist to get your POS operational:

    Setup Tasks

    • [ ] Verify the system is running and accessible
    • [ ] Log in with administrator credentials
    • [ ] Change default admin password
    • [ ] Verify or create your till
    • [ ] Assign your device to a till

    Configuration Tasks

    • [ ] Review default categories or create your own
    • [ ] Add your first product with a variant
    • [ ] Verify tax settings

    Operations Tasks

    • [ ] Take your first test order
    • [ ] Process a test payment
    • [ ] Create a test tab (optional)

    Step 1: First Login

    Access the POS application and log in for the first time:

    Accessing the Application

    1. Open your web browser (Chrome recommended)
    2. Navigate to the POS application URL (e.g., http://192.168.1.241:80)
    3. The login screen will appear

    Logging In

    1. In the Username field, enter: admin
    2. In the Password field, enter: admin123
    3. Click the Login button
    👤

    Till Assignment (First Time Only)

    If this is the first time accessing the POS on this device and no till is assigned:

    • As Administrator: You will see the Till Setup screen to select a till
    • As Cashier: You will see an error message - an admin must configure the till first
    Language Selection
    Use the language switcher (EN/IT) in the top-right corner of the login screen to change the interface language.

    Step 2: Understanding the Interface

    After logging in, you'll see the main POS interface:

    Interface Layout

    Area Location Description
    Product Grid Left/Main area Visual grid of products organized by category
    Category Tabs Top of grid Switch between product categories
    Order Panel Right side Current order items, quantities, and total
    Admin Button Top right Access admin panel (admins only)

    Admin Panel Access

    If you're logged in as an administrator, you'll see a purple Admin button in the top-right corner. Click this to access the administration panel for configuration.

    Step 3: Create Your First Category

    Categories organize your products in the POS interface. Create categories that match your menu structure:

    1. Click the Admin button (top right)
    2. In the admin panel, navigate to Categories
    3. Click Add Category
    4. Enter a category name (e.g., "Appetizers", "Main Courses", "Beverages")
    5. Select which tills should display this category:
      • Leave empty to show on all tills
      • Select specific tills to limit visibility
    6. Click Save
    Tip
    The system comes with pre-seeded categories (Red Wine, Beer, Cocktails, Whiskey, Soft Drinks). You can edit or delete these to match your menu.
    📷

    Step 4: Add Your First Product

    Products are the items you sell. Each product can have multiple variants (sizes, options):

    1. In the admin panel, navigate to Products
    2. Click Add Product
    3. Enter the product name (e.g., "Margherita Pizza")
    4. Select the category from the dropdown
    5. Add at least one variant:
      • Click Add Variant
      • Enter variant name (e.g., "Regular", "Large")
      • Set the price
      • Choose button colors (optional - helps with quick identification)
      • Mark as "Favourite" to feature it prominently (optional)
    6. Click Save

    Understanding Variants

    Variants represent different versions of the same product:

    • Size variations: Small, Medium, Large
    • Serving style: Glass, Bottle, Draft
    • Customization: Regular, Decaf, Extra Shot
    Favourites
    Mark frequently ordered variants as "Favourite" to display them in the Favourites category, making them quickly accessible during busy periods.

    Step 5: Take Your First Order

    Now you're ready to take an order. Return to the POS view:

    1. Click Back to POS or close the admin panel
    2. You'll see the product grid with your categories
    3. Click a category tab to view products in that category
    4. Click a product variant button to add it to the order
    5. The item appears in the Order Panel on the right

    Adjusting Quantities

    In the Order Panel, you can modify the current order:

    • Increase quantity: Click the + button next to an item
    • Decrease quantity: Click the - button (removes item at 0)
    • Remove item: Click the trash icon or reduce quantity to 0
    • Clear entire order: Click the Clear button (requires confirmation)

    Viewing Order Total

    The Order Panel shows:

    • Subtotal (sum of all items)
    • Tax amount (based on your tax settings)
    • Total (subtotal + tax)

    Step 6: Process Your First Payment

    Complete the sale by processing payment:

    1. Review the order total in the Order Panel
    2. Click the Pay button
    3. The Payment Modal will appear
    4. Select the payment method:
      • Cash - For cash payments
      • Card - For credit/debit card payments
    5. Add a tip if applicable (use the tip buttons or enter custom amount)
    6. Click Complete Payment
    7. The transaction is recorded and the order is cleared

    Payment Modal Features

    Feature Description
    Order Summary Shows all items being purchased
    Subtotal/Tax/Total Breakdown of the order cost
    Tip Buttons Quick tip percentages (10%, 15%, 20%) or custom amount
    Payment Method Cash or Card selection

    Additional Quick Features

    Assigning Orders to Tables

    For table service, assign orders to specific tables:

    1. In the Order Panel, click the Table button
    2. Select the room from the dropdown
    3. Click on an available table
    4. The order is now associated with that table

    Creating Tabs

    Save orders for later payment (useful for bar tabs):

    1. Add items to the order
    2. Click Tabs in the Order Panel
    3. Click Create New Tab
    4. Enter a name for the tab (e.g., customer name)
    5. The tab is saved and can be loaded later

    Using the Virtual Keyboard

    On touch devices, the virtual keyboard appears automatically when input fields are focused:

    • Numeric Keypad - For entering prices and quantities
    • Full Keyboard - For text entry (names, notes)
    • Done Button - Closes the keyboard

    Keyboard Shortcuts

    Speed up your workflow with these keyboard shortcuts:

    Shortcut Action Context

    Ctrl++ Add new table/room Table Management
    Ctrl+D Toggle layout mode Table Management
