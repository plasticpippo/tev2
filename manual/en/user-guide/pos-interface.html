    POS Interface Overview

    Get familiar with the main POS screen layout and its components.

    Main Screen Layout

    The POS interface is divided into several key areas that work together to help you process orders efficiently:

    Screen Sections

    • Header Area (top-right) - Admin button for administrators only
    • Product Grid Area (left/center) - Displays products organized by category
    • Category Tabs (above product grid) - Navigate between product categories
    • Order Panel (right side) - Shows current order items, totals, and action buttons

    Product Grid Area

    The product grid is the main working area where you select items for orders. It displays:

    Product Buttons

    • Each product appears as a large, tappable button
    • Shows the product name and price
    • Custom button colors configured by your administrator
    • Clicking a product adds it directly to the current order

    Favourites Category

    The first category tab is Favourites, which shows frequently ordered items for quick access. Your administrator can configure which products appear here.

    Category Navigation

    Category tabs allow you to switch between different groups of products:

    1. Favourites - Quick access to frequently ordered items (first tab)
    2. Category tabs - Additional categories configured by your administrator
    3. Click any category tab to display its products in the grid
    4. Scroll horizontally if all category tabs don't fit on screen
    Tip
    Use the Favourites category for your fastest-moving items to reduce the number of clicks needed for common orders.

    Order Panel

    The order panel (on the right side of the screen) is your command center for managing the current order. It displays:

    User Information Section (Top)

    • Your logged-in username
    • Your role (Admin or Cashier)
    • Logout button

    Order Information (Middle)

    • Current order items with quantities
    • Item prices
    • Active tab name (if working with a saved tab)
    • Assigned table (if table is assigned)

    Totals Section

    • Subtotal (before tax)
    • Total amount including tax

    Action Buttons (Bottom)

    • View Open Tabs / Tabs - Opens the tab manager
    • Clear Order - Removes all items from current order
    • Assign Table - Opens table assignment modal
    • Payment - Opens the payment modal

    Header Controls

    The top-right area of the POS interface contains additional controls:

    • Admin Button - Visible only to administrators; opens the admin management panel
    • Language Switcher - Located on the login screen; allows switching between English and Italian
    Note
    The Admin button is only visible to users with Administrator role. Cashiers will not see this button.

    Layout Customization

    Administrators can customize how products appear in the grid:

    • Product button colors
    • Button sizes and positions
    • Product groupings
    • Category assignments

    To access layout editing, administrators click the Edit Layout button in the order panel (visible only to admins).
