    Virtual Keyboard

    Learn how to use the on-screen virtual keyboard for touchscreen devices.

    When the Keyboard Appears

    The virtual keyboard appears automatically when you tap on certain input fields:

    • Text input fields - Like username, password, or tab name
    • Number input fields - Like tip amounts or quantities
    • Any editable text area - Where text entry is needed
    Tip
    The keyboard appears when you tap an input field and disappears when you tap outside of it or press Done.

    Keyboard Types

    The POS uses two different keyboard layouts:

    Numeric Keypad

    For number-only input (like tip amounts), a simplified numeric keypad appears:

    • Numbers 0-9
    • Enter key
    • Backspace (delete) key
    • Done button to close keyboard

    Special Keys

    The full keyboard includes several special keys:

    Key Symbol Function
    Shift ↑ (up arrow) Toggle capital letters for one keystroke
    Caps Lock ⇪ (up arrow with line) Toggle all capital letters mode
    Backspace ⌫ (delete symbol) Delete the last character entered
    Space space Insert a space between words
    Enter ↵ (return arrow) Confirm input or move to next field
    Done Done Close the virtual keyboard

    Keyboard Positioning

    The virtual keyboard automatically positions itself:

    • Below the input field - When there's enough space
    • Above the input field - When there's not enough space below
    • Centered horizontally - Aligns with the input field
    • Adapts to screen size - Works on various device sizes
    Note
    The keyboard tries to avoid covering the input field, but you may need to scroll the page to see what you're typing if the field is near the bottom of the screen.

    How to Use the Keyboard

    Typing Text

    1. Tap on the input field where you want to type
    2. The virtual keyboard appears
    3. Tap letters to enter text
    4. Use Shift for capital letters
    5. Use Backspace to correct mistakes
    6. Press Done when finished

    Typing Numbers

    1. Tap on a number field
    2. The numeric keypad appears
    3. Tap numbers to enter values
    4. Use Backspace to delete
    5. Press Done or tap outside to close

    Enabling/Disabling Virtual Keyboard

    The virtual keyboard can be toggled on or off:

    For Administrators

    Administrators can control the virtual keyboard setting in the Admin Panel:

    1. Click the Admin button
    2. Go to Settings
    3. Find the virtual keyboard option
    4. Enable or disable as needed
    Note
    The virtual keyboard is helpful for touchscreen devices but may not be needed when using a physical keyboard. Disabling it can free up screen space.
