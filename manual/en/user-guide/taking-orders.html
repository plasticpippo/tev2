    Taking Orders

    Learn how to browse products, add items to cart, adjust quantities, and manage orders.

    Browsing Products by Category

    Products are organized into categories for easy navigation. To browse and select products:

    1. Look at the category tabs above the product grid
    2. The first tab is Favourites - showing frequently ordered items
    3. Click on any category tab to view its products
    4. Products in the selected category will appear in the grid
    5. Scroll through the grid if products don't fit on one screen

    Adding Items to Order

    To add a product to the current order:

    1. Navigate to the appropriate category using the category tabs
    2. Find the desired product in the grid
    3. Click on the product button
    4. The item immediately appears in the order panel with quantity 1
    5. Click the same product again to increase the quantity

    Quick Quantity Increase

    Repeatedly clicking the same product button quickly adds multiple quantities. Each click adds one more of that item to the order.

    Adjusting Quantities

    Use the quantity controls in the order panel to modify item quantities:

    Increase Quantity

    • Click the + (plus) button next to an item
    • Or click the product button in the grid again

    Decrease Quantity

    • Click the - (minus) button next to an item
    • When quantity reaches 0, the item is removed from the order
    Important
    Decreasing quantity to 0 removes the item from the order entirely. This action cannot be undone.

    Viewing Order Totals

    The order panel displays running totals as you build the order:

    • Order Items - Each item shows name, unit price, and quantity
    • Subtotal - The sum of all items before tax
    • Tax - Automatically calculated based on tax settings
    • Total - The final amount including tax

    The total updates automatically as you add, remove, or modify items.

    Clearing the Order

    To remove all items from the current order:

    1. Click the Clear Order button in the order panel
    2. Confirm the action when prompted
    3. All items are removed and the order starts fresh
    Warning
    Clearing an order permanently removes all items. This action cannot be undone. Make sure you want to clear all items before proceeding.
    Alternative
    Instead of clearing, consider saving the order as a Tab if the customer might return or want to pay later.

    Order Workflow Summary

    Here is the typical workflow for taking an order:

    1. Start a new order - The order panel should be empty (or load an existing tab)
    2. Select category - Click the appropriate category tab
    3. Add items - Click products to add them to the order
    4. Adjust quantities - Use +/- buttons as needed
    5. Assign table (optional) - Click "Assign Table" to link order to a table
    6. Review totals - Check subtotal, tax, and total
    7. Process payment - Click "Payment" when ready to checkout
