    Working with Tabs

    Learn how to create, save, load, and manage open tabs for ongoing orders.

    What Are Tabs?

    Tabs are saved orders that you can keep open for later. They are useful when:

    • Customers want to order more items later
    • A table has multiple customers who pay separately
    • You need to step away but want to preserve the order
    • Customers are split-checking during a meal
    Note
    Tabs persist in the system even after you log out. The next staff member can access and continue working on any open tabs.

    Creating New Tabs

    To create a new tab:

    1. Build your order as usual by adding products
    2. Click Save Tab in the order panel
    3. Enter a name for the tab (e.g., "Table 5", "John's Tab")
    4. Click Create or press Enter
    5. The order is saved as a tab and the current order clears

    Loading Existing Tabs

    To continue working on a saved tab:

    1. Click Tabs button in the order panel
    2. The Tab Manager modal opens showing all open tabs
    3. Find the tab you want to load
    4. Click Load Tab next to that tab
    5. The order items restore in the order panel
    Tip
    Tabs are sorted by creation time, with the newest tabs appearing first. Look for your tab by name or scroll through the list.

    Adding Items to Existing Tabs

    To add items to a tab without loading it:

    1. Build your current order with new items
    2. Click Tabs in the order panel
    3. Find the target tab
    4. Click Add to Tab next to that tab
    5. The items are added to the tab and your current order clears
    Important
    You can only add items to tabs if you have items in your current order. Build the order first, then use "Add to Tab".

    Closing Tabs

    To close (and delete) a tab:

    1. Click Tabs in the order panel
    2. Find the tab you want to close
    3. Click Load Tab to load it first
    4. Process the payment (click Payment)
    5. After payment, the tab is automatically closed

    Alternatively, if a tab has no items:

    1. Click Tabs in the order panel
    2. Find the empty tab
    3. Click Close Tab to remove it

    Transferring Items Between Tabs

    To move items from one tab to another:

    1. Click Tabs in the order panel
    2. Find the source tab (the tab items are coming from)
    3. Click Transfer next to that tab
    4. The Transfer Items modal opens
    5. Select the items you want to transfer
    6. Choose the destination tab
    7. Click Transfer to complete

    Tab Information Display

    The Tab Manager shows for each tab:

    • Tab name - The name you assigned when creating
    • Tab total - The sum of all items in the tab
    • Table assignment - Which table the tab is linked to (if any)
    • Creation date - When the tab was created

    Tabs and Tables

    Tabs can be linked to tables:

    • When you assign a table to an order, then save as a tab, the table is remembered
    • When you load a tab with a table, the table assignment restores
    • This helps track which tab belongs to which table
