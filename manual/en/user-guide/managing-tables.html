    Managing Tables

    Learn how to assign orders to tables, view table status, and manage table assignments.

    Assigning Orders to Tables

    Linking orders to tables helps track where customers are seated and organize the workflow. To assign an order to a table:

    1. Build your order as usual by adding products
    2. Click the Assign Table button in the order panel
    3. The Table Assignment modal opens showing a visual layout
    4. Select a Room from the room tabs (if multiple rooms exist)
    5. Choose an available table from the visual layout
    6. Click Assign to Table to confirm

    Table Status Meanings

    Each table has a status that indicates its current condition. The status is shown by color-coded borders:

    Status Meaning Color Selectable
    Available Table is free and ready for new customers Green Yes
    Occupied Table has active orders Red No
    Reserved Table is reserved for upcoming guests Yellow Yes
    Unavailable Table is out of service Gray No
    Note
    Only tables with "Available" or "Reserved" status can be assigned to new orders. Occupied and unavailable tables cannot be selected.

    Changing Table Assignments

    To change which table an order is assigned to:

    1. Click the Assign Table button (or "Change Table" if already assigned)
    2. Select a different available table
    3. Click Assign to Table to confirm

    To remove the table assignment entirely:

    1. Click Assign Table button
    2. Click Clear Table to remove the assignment

    Navigating Rooms

    If your establishment has multiple rooms (e.g., Main Dining, Patio, Private), use the room selector:

    1. In the Table Assignment modal, click on the room name tab
    2. The table layout updates to show tables in that room
    3. Select a table from the current room view
    Tip
    The visual layout shows tables in their actual positions within each room, making it easy to identify the right table at a glance.

    Table Indicators in Order Panel

    When a table is assigned to the current order, the order panel displays:

    • Table name - The assigned table number/name
    • Table status - Current status (available, occupied, etc.)
    • Change Table button - Quick access to reassign

    Tables and Tabs

    Tables can be combined with tabs for better workflow:

    • Save an order as a Tab to keep it open for later
    • The tab remembers which table it was assigned to
    • When you load the tab later, the table assignment is restored
    • Process payment when the customer is ready to leave
