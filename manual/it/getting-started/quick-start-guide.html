    Guida Rapida

    Inizia rapidamente con il sistema POS. Questa guida ti accompagna attraverso i passaggi essenziali per iniziare a prendere ordini ed elaborare pagamenti in pochi minuti.

    Checklist Guida Rapida

    Segui questa checklist per rendere operativo il tuo POS:

    Attività di Configurazione

    • [ ] Verifica che il sistema sia in funzione e accessibile
    • [ ] Accedi con le credenziali amministratore
    • [ ] Cambia la password admin predefinita
    • [ ] Verifica o crea la tua cassa
    • [ ] Assegna il tuo dispositivo a una cassa

    Attività di Configurazione

    • [ ] Rivedi le categorie predefinite o creane di tue
    • [ ] Aggiungi il tuo primo prodotto con una variante
    • [ ] Verifica le impostazioni delle tasse

    Attività Operative

    • [ ] Prendi il tuo primo ordine di test
    • [ ] Elabora un pagamento di test
    • [ ] Crea un conto di test (opzionale)

    Passaggio 1: Primo Accesso

    Accedi all'applicazione POS e accedi per la prima volta:

    Accedere all'Applicazione

    1. Apri il tuo browser web (Chrome consigliato)
    2. Naviga all'URL dell'applicazione POS (es. http://192.168.1.241:80)
    3. Apparirà la schermata di accesso

    Effettuare l'Accesso

    1. Nel campo Nome Utente, inserisci: admin
    2. Nel campo Password, inserisci: admin123
    3. Clicca il pulsante Accedi
    👤
    Screenshot: Schermata di Accesso

    Assegnazione Cassa (Solo Prima Volta)

    Se è la prima volta che accedi al POS su questo dispositivo e nessuna cassa è assegnata:

    • Come Amministratore: Vedrai la schermata Configurazione Cassa per selezionare una cassa
    • Come Cassiere: Vedrai un messaggio di errore - un admin deve prima configurare la cassa
    Selezione Lingua
    Usa il selettore lingua (EN/IT) nell'angolo in alto a destra della schermata di accesso per cambiare la lingua dell'interfaccia.

    Accesso al Pannello Admin

    Se hai effettuato l'accesso come amministratore, vedrai un pulsante viola Admin nell'angolo in alto a destra. Clicca questo per accedere al pannello di amministrazione per la configurazione.

    Passaggio 3: Crea la Tua Prima Categoria

    Le categorie organizzano i tuoi prodotti nell'interfaccia POS. Crea categorie che corrispondono alla struttura del tuo menu:

    1. Clicca il pulsante Admin (in alto a destra)
    2. Nel pannello admin, vai a Categorie
    3. Clicca Aggiungi Categoria
    4. Inserisci un nome categoria (es. "Antipasti", "Primi Piatti", "Bevande")
    5. Seleziona quali casse devono mostrare questa categoria:
      • Lascia vuoto per mostrare su tutte le casse
      • Seleziona casse specifiche per limitare la visibilità
    6. Clicca Salva
    Consiglio
    Il sistema viene fornito con categorie predefinite (Vino Rosso, Birra, Cocktail, Whiskey, Bevande Analcoliche). Puoi modificare o eliminare queste per corrispondere al tuo menu.

    Passaggio 4: Aggiungi il Tuo Primo Prodotto

    I prodotti sono gli articoli che vendi. Ogni prodotto può avere più varianti (dimensioni, opzioni):

    1. Nel pannello admin, vai a Prodotti
    2. Clicca Aggiungi Prodotto
    3. Inserisci il nome del prodotto (es. "Pizza Margherita")
    4. Seleziona la categoria dal menu a tendina
    5. Aggiungi almeno una variante:
      • Clicca Aggiungi Variante
      • Inserisci il nome variante (es. "Normale", "Large")
      • Imposta il prezzo
      • Scegli i colori dei pulsanti (opzionale - aiuta per l'identificazione rapida)
      • Contrassegna come "Preferito" per visualizzarla in evidenza (opzionale)
    6. Clicca Salva

    Comprensione delle Varianti

    Le varianti rappresentano diverse versioni dello stesso prodotto:

    • Variazioni di dimensione: Piccolo, Medio, Grande
    • Stile di servizio: Bicchiere, Bottiglia, Spina
    • Personalizzazione: Normale, Decaffeinato, Extra Shot
    Preferiti
    Contrassegna le varianti ordinate frequentemente come "Preferito" per visualizzarle nella categoria Preferiti, rendendole rapidamente accessibili durante i periodi di lavoro intenso.

    Passaggio 5: Prendi il Tuo Primo Ordine

    Ora sei pronto per prendere un ordine. Torna alla vista POS:

    1. Clicca Torna al POS o chiudi il pannello admin
    2. Vedrai la griglia prodotti con le tue categorie
    3. Clicca una scheda categoria per visualizzare i prodotti in quella categoria
    4. Clicca un pulsante variante prodotto per aggiungerlo all'ordine
    5. L'articolo appare nel Pannello Ordini sulla destra

    Regolazione delle Quantità

    Nel Pannello Ordini, puoi modificare l'ordine corrente:

    • Aumenta quantità: Clicca il pulsante + accanto a un articolo
    • Diminuisci quantità: Clicca il pulsante - (rimuove l'articolo a 0)
    • Rimuovi articolo: Clicca l'icona del cestino o riduci la quantità a 0
    • Cancella ordine intero: Clicca il pulsante Cancella (richiede conferma)

    Visualizzazione Totale Ordine

    Il Pannello Ordini mostra:

    • Subtotale (somma di tutti gli articoli)
    • Importo tasse (in base alle tue impostazioni delle tasse)
    • Totale (subtotale + tasse)

    Passaggio 6: Elabora il Tuo Primo Pagamento

    Completa la vendita elaborando il pagamento:

    1. Rivedi il totale dell'ordine nel Pannello Ordini
    2. Clicca il pulsante Paga
    3. Apparirà il Modale Pagamento
    4. Seleziona il metodo di pagamento:
      • Contanti - Per pagamenti in contanti
      • Carta - Per pagamenti con carta di credito/debito
    5. Aggiungi una mancia se applicabile (usa i pulsanti mancia o inserisci un importo personalizzato)
    6. Clicca Completa Pagamento
    7. La transazione viene registrata e l'ordine viene cancellato

    Funzionalità del Modale Pagamento

    Funzionalità Descrizione
    Riepilogo Ordine Mostra tutti gli articoli acquistati
    Subtotale/Tasse/Totale Ripartizione del costo dell'ordine
    Pulsanti Mancia Percentuali mancia rapide (10%, 15%, 20%) o importo personalizzato
    Metodo Pagamento Selezione Contanti o Carta

    Funzionalità Rapide Aggiuntive

    Assegnazione Ordini ai Tavoli

    Per il servizio al tavolo, assegna ordini a tavoli specifici:

    1. Nel Pannello Ordini, clicca il pulsante Tavolo
    2. Seleziona la sala dal menu a tendina
    3. Clicca su un tavolo disponibile
    4. L'ordine è ora associato a quel tavolo

    Creazione Conti

    Salva ordini per pagamento successivo (utile per conti al bar):

    1. Aggiungi articoli all'ordine
    2. Clicca Conti nel Pannello Ordini
    3. Clicca Crea Nuovo Conto
    4. Inserisci un nome per il conto (es. nome cliente)
    5. Il conto viene salvato e può essere caricato successivamente

    Utilizzo della Tastiera Virtuale

    Sui dispositivi touch, la tastiera virtuale appare automaticamente quando i campi di input sono focalizzati:

    • Tastierino Numerico - Per inserire prezzi e quantità
    • Tastiera Completa - Per inserimento testo (nomi, note)
    • Pulsante Fatto - Chiude la tastiera

    Scorciatoie da Tastiera

    Velocizza il tuo lavoro con queste scorciatoie da tastiera:
