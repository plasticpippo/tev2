    Accesso

    Impara come accedere al sistema POS, comprende l'assegnazione delle casse e risolvi i problemi comuni di accesso.

    Panoramica Schermata Accesso

    La schermata di accesso è il punto di ingresso al sistema POS. Quando apri l'applicazione, vedrai:

    • Campo nome utente - Inserisci il tuo nome utente assegnato
    • Campo password - Inserisci la tua password (i caratteri sono nascosti per sicurezza)
    • Pulsante Accedi - Clicca per autenticarti e accedere al sistema
    • Selettore lingua - Passa tra Inglese e Italiano nell'angolo in alto a destra
    • Informazioni cassa - Visualizza il nome della cassa assegnata a questo dispositivo

    Inserire le Tue Credenziali

    Per accedere al sistema POS:

    1. Inserisci il tuo nome utente - Tocca o clicca sul campo nome utente e digita il tuo nome utente assegnato
    2. Inserisci la tua password - Tocca o clicca sul campo password e digita la tua password
    3. Clicca Accedi - Premi il pulsante Accedi o premi Invio sulla tastiera virtuale
    Utilizzo della Tastiera Virtuale
    Sui dispositivi touchscreen, toccare un campo di input apre automaticamente la tastiera virtuale. Puoi usare sia la tastiera virtuale che una tastiera fisica per inserire le tue credenziali. La tastiera fornisce un layout QWERTY completo per l'inserimento di testo.

    Comprensione dell'Assegnazione Cassa

    Ogni dispositivo POS deve essere assegnato a una cassa specifica (registratore di cassa). Il nome della cassa viene visualizzato sulla schermata di accesso sotto il titolo.

    Indicatori di Stato Cassa

    • Testo bianco - La cassa è configurata correttamente e pronta per l'uso
    • Testo rosso - Nessuna cassa assegnata a questo dispositivo; solo gli amministratori possono accedere

    Per Amministratori: Selezionare una Cassa

    Se nessuna cassa è assegnata e sei un amministratore, vedrai la schermata Configurazione Cassa dopo l'accesso:

    1. Rivedi l'elenco delle casse disponibili
    2. Clicca sul nome della cassa per selezionarla
    3. Il sistema assegnerà questo dispositivo alla cassa selezionata

    Risoluzione Errori di Accesso

    Se riscontri problemi nell'accesso, controlla le seguenti soluzioni:

    Messaggio di Errore Causa Soluzione
    Credenziali non valide Nome utente o password errati Controlla le credenziali per errori di battitura. Assicurati che il blocco maiuscole sia disattivato. Riprova attentamente.
    Cassa non configurata Nessuna cassa assegnata a questo dispositivo Contatta il tuo amministratore per assegnare una cassa a questo dispositivo prima di poter accedere.
    Credenziali mancanti Il campo nome utente o password è vuoto Riempi entrambi i campi nome utente e password prima di cliccare Accedi.
    Errore server Il server backend non risponde Controlla la tua connessione di rete. Contatta il supporto tecnico se il problema persiste.
    Accesso non riuscito Errore generale di autenticazione Verifica le tue credenziali e riprova. Se il problema continua, contatta il tuo amministratore.

    Gestione Sessione

    Rimani Connesso

    Una volta effettuato l'accesso, la tua sessione rimane attiva fino a quando non ti disconnetti esplicitamente. Il sistema mantiene il tuo stato di accesso anche se aggiorni la page o perdi temporaneamente la connettività di rete.

    Disconnessione

    Per disconnettersi dal sistema POS:

    1. Individua il tuo nome utente visualizzato nel pannello ordini (lato destro dello schermo)
    2. Clicca il pulsante Disconnetti accanto al tuo nome
    3. Verrai riportato alla schermata di accesso
    Consiglio
    Disconnettiti sempre quando lasci il terminale POS incustodito. Questo previene transazioni non autorizzate e garantisce la corretta tracciabilità per tutti gli ordini.

    Best Practice di Sicurezza

    • Non condividere mai la tua password - Ogni membro del personale dovrebbe avere il proprio account
    • Disconnettiti quando ti allontani - Disconnettiti sempre quando ti allontani dal terminale
    • Segnala attività sospette - Informi immediatamente il tuo amministratore se noti qualcosa di insolito
    • Cambia le password regolarmente - Chiedi al tuo amministratore di aggiornare la tua password periodicamente
    • Controlla il tuo nome utente - Verifica che il tuo nome appaia nel pannello ordini dopo l'accesso

    Ruoli Utente

    Il sistema POS ha due tipi di utenti:

    Ruolo Descrizione Funzionalità Aggiuntive
    Admin Accesso completo al sistema per manager e supervisori Pannello Admin, Gestione Utenti, Report, Impostazioni, Configurazione Cassa
    Cassiere Accesso standard per il personale di sala Operazioni POS, presa ordini, elaborazione pagamenti

    Gli amministratori vedranno un pulsante Admin nell'angolo in alto a destra della schermata POS principale, che fornisce accesso al pannello di gestione.
