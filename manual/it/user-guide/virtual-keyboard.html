    Tastiera Virtuale

    Impara come utilizzare la tastiera virtuale su schermo per dispositivi touchscreen.

    Quando Appare la Tastiera

    La tastiera virtuale appare automaticamente quando tocchi certi campi di input:

    • Campi di input testo - Come nome utente, password o nome conto
    • Campi di input numerico - Come importi mancia o quantità
    • Qualsiasi area di testo modificabile - Dove è necessario l'inserimento di testo
    Consiglio
    La tastiera appare quando tocchi un campo di input e scompare quando tocchi fuori da esso o premi Fatto.

    Tipi di Tastiera

    Il POS utilizza due diversi layout di tastiera:

    Tastierino Numerico

    Per input solo numerico (come importi mancia), appare un tastierino numerico semplificato:

    • Numeri 0-9
    • Pulsante Invio
    • Pulsante Backspace (cancella)
    • Pulsante Fatto per chiudere la tastiera

    Tasti Speciali

    La tastiera completa include diversi tasti speciali:

    Tasto Simbolo Funzione
    Maius ↑ (freccia su) Attiva/disattiva lettere maiuscole per un singolo tasto
    BLOCCO MAIUSC ⇪ (freccia su con linea) Attiva/disattiva modalità tutte maiuscole
    Backspace ⌫ (simbolo cancella) Cancella l'ultimo carattere inserito
    Spazio spazio Inserisci uno spazio tra le parole
    Invio ↵ (freccia ritorno) Conferma input o passa al campo successivo
    Fatto Fatto Chiude la tastiera virtuale

    Posizionamento Tastiera

    La tastiera virtuale si posiziona automaticamente:

    • Sotto il campo di input - Quando c'è spazio sufficiente
    • Sopra il campo di input - Quando non c'è spazio sufficiente sotto
    • Centrata orizzontalmente - Si allinea con il campo di input
    • Si adatta alle dimensioni dello schermo - Funziona su varie dimensioni di dispositivo
    Nota
    La tastiera cerca di non coprire il campo di input, ma potresti dover scorrere la pagina per vedere ciò che stai digitando se il campo è vicino alla parte inferiore dello schermo.

    Come Usare la Tastiera

    Digitare Testo

    1. Tocca sul campo di input dove vuoi digitare
    2. Appare la tastiera virtuale
    3. Tocca le lettere per inserire il testo
    4. Usa Maius per lettere maiuscole
    5. Usa Backspace per correggere errori
    6. Premi Fatto quando hai finito

    Digitare Numeri

    1. Tocca su un campo numerico
    2. Appare il tastierino numerico
    3. Tocca i numeri per inserire i valori
    4. Usa Backspace per cancellare
    5. Premi Fatto o tocca fuori per chiudere

    Abilitare/Disabilitare la Tastiera Virtuale

    La tastiera virtuale può essere attivata o disattivata:

    Per Amministratori

    Gli amministratori possono controllare l'impostazione della tastiera virtuale nel Pannello Admin:

    1. Clicca il pulsante Admin
    2. Vai a Impostazioni
    3. Trova l'opzione tastiera virtuale
    4. Abilita o disabilita come necessario
    Nota
    La tastiera virtuale è utile per dispositivi touchscreen ma potrebbe non essere necessaria quando si usa una tastiera fisica. Disabilitarla può liberare spazio sullo schermo.
