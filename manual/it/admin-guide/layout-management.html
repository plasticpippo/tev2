    Gestione Layout

    La gestione dei layout ti permette di personalizzare come i prodotti appaiono sullo schermo POS di ogni cassa. Crea disposizioni personalizzate, condividi layout tra le casse e ottimizza la griglia prodotti per un servizio efficiente.

    Panoramica

    I layout determinano la disposizione visiva dei prodotti nell'interfaccia POS. Ogni cassa puo' avere il proprio layout, oppure puoi creare layout condivisi che possono essere applicati a piu' casse.

    Tipi di Layout

    Tipo Descrizione Caso d'Uso
    Layout Specifico per Cassa Layout personalizzato per una singola cassa Configurazioni uniche per terminale
    Layout Condiviso Layout riutilizzabile che puo' essere applicato a piu' casse Layout standard tra terminali simili

    Creazione di Layout Specifici per Cassa

    Ogni cassa puo' avere il proprio layout personalizzato per ogni categoria. Il layout determina dove appare ogni variante del prodotto nella griglia.

    Accesso all'Editor di Layout

    1. Accedi al POS sulla cassa di destinazione
    2. Entra in Modalita' Amministratore
    3. Naviga alla categoria che vuoi organizzare
    4. Premi a lungo o fai clic destro sulla griglia prodotti
    5. Seleziona Modifica Layout o usa l'interfaccia di editing del layout

    Disposizione dei Prodotti

    1. Trascina i prodotti per riposizionarli nella griglia
    2. I prodotti si agganciano a una griglia a 4 colonne
    3. Possono essere usate multiple righe
    4. Il layout si salva automaticamente

    Configurazione della Griglia

    • Colonne: Fisse a 4 colonne
    • Righe: Illimitate - scorri secondo necessita'
    • Posizione: Specificata da gridColumn (1-4) e gridRow (a partire da 1)

    Creazione di Layout Condivisi

    I layout condivisi ti permettono di creare una disposizione standard una sola volta e applicarla a piu' casse.

    Creazione di un Layout Condiviso

    1. Naviga a Gestione Layout nel Pannello di Amministrazione
    2. Clicca su Crea Layout Condiviso
    3. Inserisci un Nome per il layout
    4. Seleziona la Categoria a cui si applica questo layout
    5. Aggiungi i prodotti e specifica le loro posizioni
    6. Clicca su Salva

    Opzioni del Layout Condiviso

    • Nome: Identificatore descrittivo per il layout
    • Categoria: Quale categoria di prodotti organizza questo layout
    • Posizioni: Mappature variante-prodotto-posizione-griglia
    • Proprietario: Utente che ha creato il layout

    Applicazione dei Layout Condivisi

    Dopo aver creato un layout condiviso, puoi applicarlo a qualsiasi cassa.

    Caricamento di un Layout su una Cassa

    1. Naviga a Layout Condivisi nel Pannello di Amministrazione
    2. Trova il layout che vuoi applicare
    3. Clicca su Carica su Cassa o usa l'azione "Carica su Cassa"
    4. Seleziona la cassa di destinazione dalla lista
    5. Conferma l'azione
    Nota: Caricare un layout condiviso sostituisce qualsiasi layout esistente per quella combinazione cassa-categoria. Il layout viene copiato, non collegato, quindi le modifiche al layout condiviso non influenzeranno le casse dove e' gia' stato caricato.

    Modifica dei Layout

    Puoi modificare i layout esistenti in qualsiasi momento.

    Modifica dei Layout Specifici per Cassa

    1. Accedi al POS sulla cassa di destinazione
    2. Naviga alla categoria
    3. Entra in modalita' modifica layout
    4. Trascina i prodotti in nuove posizioni
    5. Le modifiche si salvano automaticamente

    Modifica dei Layout Condivisi

    1. Naviga a Layout Condivisi nel Pannello di Amministrazione
    2. Clicca su Modifica sul layout
    3. Modifica la configurazione del layout
    4. Clicca su Salva

    Eliminazione dei Layout

    Rimuovi i layout quando non sono piu' necessari.

    Eliminazione di un Layout di Cassa

    1. Accedi al POS sulla cassa di destinazione
    2. Naviga alla categoria
    3. Entra in modalita' modifica layout
    4. Seleziona Reimposta Layout o Cancella Layout
    5. Conferma l'azione

    Eliminazione di un Layout Condiviso

    1. Naviga a Layout Condivisi nel Pannello di Amministrazione
    2. Clicca su Elimina sul layout
    3. Conferma l'eliminazione
    Avviso: Eliminare un layout condiviso non influenza le casse dove e' gia' stato caricato. Quelle casse manterranno la loro copia del layout.

    Categoria Preferiti

    La categoria Preferiti e' una pseudo-categoria speciale che mostra tutti i prodotti marcati come preferiti, indipendentemente dalla categoria regolare a cui appartengono.

    Creazione del Layout Preferiti

    1. I prodotti devono essere marcati come preferiti nelle impostazioni delle varianti
    2. La categoria Preferiti (ID: -1) appare nella lista categorie del POS
    3. Crea un layout per la categoria Preferiti come per qualsiasi altra

    Layout per Preferiti

    • Tutti i prodotti preferiti appaiono in questa categoria
    • Puoi personalizzare la loro disposizione
    • Utile per accesso rapido agli articoli popolari

    Best Practices

    Progettazione del Layout

    • Posiziona gli articoli piu' ordinati nelle righe superiori
    • Raggruppa i prodotti correlati
    • Considera la zona di raggiungibilita' per i terminali touch-screen
    • Usa posizioni consistenti tra casse simili

    Layout Condivisi

    • Crea layout condivisi per configurazioni terminale standard
    • Usa layout diversi per tipi di terminale diversi (es. bar vs. bancone)
    • Documenta quali casse usano quali layout
    • Aggiorna i layout condivisi quando cambiano i prodotti

    Manutenzione

    • Rivedi i layout quando aggiungi nuovi prodotti
    • Rimuovi i layout per prodotti discontinuati
    • Testa i layout effettuando l'accesso su ogni cassa
    • Considera il feedback del personale sull'efficienza del layout
    Suggerimento: Usa i layout condivisi per mantenere la consistenza tra terminali simili, poi apporta piccole aggiustamenti per locali specifici secondo necessita'.

    Risoluzione Problemi

    Prodotti che Non Appare

    Se i prodotti non appaiono nella griglia POS:

    • Verifica che la categoria sia visibile per quella cassa
    • Controlla che il prodotto abbia almeno una variante
    • Assicurati che il prodotto sia in una categoria attiva

    Il Layout non si Salva

    Se le modifiche al layout non si salvano:

    • Controlla la connettivita' di rete
    • Assicurati di essere loggato con i permessi appropriati
    • Prova ad aggiornare la pagina

    Il Layout Condiviso non si Applica

    Se un layout condiviso non si applica a una cassa:

    • Verifica di star caricando sulla cassa corretta
    • Controlla che la categoria del layout corrisponda alla categoria POS
    • La cassa potrebbe avere un layout specifico che sovrascrive quello condiviso
