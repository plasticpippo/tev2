    Gestione Utenti

    Gli account utente permettono al personale di accedere al sistema POS. Questa guida spiega come creare, modificare e gestire gli account utente, inclusa l'assegnazione dei ruoli e la gestione delle password.

    Visualizzazione di Tutti gli Utenti

    Naviga alla sezione Utenti dalla sidebar del Pannello Amministratore per vedere l'elenco completo di tutti gli account utente nel tuo sistema.

    Informazioni dell'Elenco Utenti

    Per ogni utente, l'elenco visualizza:

    • Nome: Il nome visualizzato dell'utente
    • Nome Utente: L'identificatore per il login
    • Ruolo: Admin o Cassiere
    • Azioni: Pulsanti modifica ed elimina

    Aggiunta di Nuovi Utenti

    Per creare un nuovo account utente:

    1. Clicca il pulsante Aggiungi Utente in cima all'elenco utenti
    2. Inserisci il Nome dell'utente (nome visualizzato mostrato nell'interfaccia)
    3. Inserisci un Nome Utente (usato per il login)
    4. Inserisci una Password per l'account
    5. Seleziona un Ruolo (Admin o Cassiere)
    6. Clicca Salva per creare l'account

    Requisiti del Nome Utente

    • Deve essere univoco (nessun utente può avere lo stesso nome utente)
    • Non fa distinzione tra maiuscole e minuscole (John e john sono considerati lo stesso)
    • Deve essere facile da ricordare per l'utente
    • Non può essere cambiato dopo la creazione (senza eliminare e ricreare)

    Requisiti della Password

    • Dovrebbe essere lunga almeno 8 caratteri
    • Dovrebbe includere una combinazione di lettere, numeri e simboli
    • Le password sono salvate in modo sicuro con hash e crittografia
    Nota di Sicurezza: Scegli password forti per gli account amministratore. Gli utenti admin hanno pieno accesso al sistema, inclusi dati finanziari sensibili.

    Ruoli e Permessi Utente

    Il sistema POS utilizza il controllo degli accessi basato sui ruoli per gestire cosa gli utenti possono fare. Esistono due ruoli principali:

    Ruolo Permessi Uso Tipico
    Admin
    • Accesso completo al Pannello Amministratore
    • Crea, modifica, elimina prodotti
    • Gestisci utenti e ruoli
    • Configura impostazioni di sistema
    • Chiudi giornata lavorativa
    • Visualizza tutti i report e analisi
    • Gestisci inventario e magazzino
    • Tutti i permessi del Cassiere
    		 Manager, proprietari, supervisori
    Cassiere
    • Accesso solo all'interfaccia POS
    • Elabora ordini e pagamenti
    • Crea e gestisci conti
    • Assegna tavoli
    • Nessun accesso al Pannello Amministratore
    		 Cassieri, camerieri, baristi

    Assegnazione dei Ruoli

    Quando crei un utente, seleziona il ruolo appropriato basato sulle responsabilità del lavoro:

    • Admin: Per utenti che devono gestire il sistema, configurare impostazioni o visualizzare report
    • Cassiere: Per utenti che devono solo elaborare vendite e gestire ordini
    Best Practice: Limita l'accesso admin solo a chi ne ha veramente bisogno. Più account admin ci sono, maggiore è il rischio di sicurezza.

    Modifica degli Account Utente

    Per modificare un account utente esistente:

    1. Individua l'utente nell'elenco utenti
    2. Clicca il pulsante Modifica (icona matita)
    3. Modifica i campi desiderati:
      • Nome: Aggiorna il nome visualizzato
      • Ruolo: Cambia tra Admin e Cassiere
      • Password: Inserisci una nuova password (opzionale)
    4. Clicca Salva per applicare le modifiche

    Cosa Può Essere Cambiato

    • Nome: Può essere cambiato in qualsiasi momento
    • Ruolo: Può essere promosso o retrocesso secondo necessità
    • Password: Può essere reimpostata (vedi sezione Gestione Password)
    • Nome Utente: Non può essere cambiato direttamente; crea un nuovo utente invece

    Gestione Password

    Le password sono un componente critico della sicurezza. Il sistema fornisce strumenti per gli amministratori per gestire le password degli utenti.

    Reimpostazione della Password di un Utente

    Come amministratore, puoi reimpostare la password di qualsiasi utente:

    1. Modifica l'account utente
    2. Inserisci una nuova password nel campo Password
    3. Salva le modifiche
    4. Comunica la nuova password all'utente in modo sicuro
    Nota: Quando reimposti una password, le sessioni esistenti dell'utente rimarranno attive fino a quando non effettueranno il logout o la sessione scadrà. Per preoccupazioni immediate sulla sicurezza, considera la revoca di tutti i token.

    Best Practice per la Sicurezza delle Password

    • Usa password forti e uniche per ogni account
    • Cambia le password periodicamente
    • Non condividere mai le password tra gli utenti
    • Cambia le password immediatamente se sospetti che siano state compromesse
    • Comunica le nuove password attraverso canali sicuri (non email o messaggi)

    Revoca delle Sessioni Utente

    Se hai bisogno di disconnettere immediatamente un utente da tutti i dispositivi:

    1. Questa azione revoca tutti i token di autenticazione per l'utente
    2. L'utente dovrà effettuare il login novamente con la password attuale
    3. Usalo se sospetti accesso non autorizzato o quando un dipendente se ne va

    Eliminazione degli Account Utente

    Per rimuovere un account utente dal sistema:

    1. Individua l'utente nell'elenco utenti
    2. Clicca il pulsante Elimina (icona cestino)
    3. Conferma l'eliminazione quando richiesto
    Prima di Eliminare: Considera quanto segue:
    • Le transazioni storiche manterranno il nome dell'utente
    • Le chiusure giornaliere faranno ancora riferimento all'utente che le ha effettuate
    • Non puoi recuperare un account eliminato

    Alternativa all'Eliminazione

    Invece di eliminare un utente, considera:

    • Cambiare il loro ruolo in Cassiere se non hanno più bisogno dell'accesso admin
    • Cambiare la loro password per impedire il login mantenendo i registri

    Visualizzazione delle Prestazioni Utente

    Il sistema traccia l'attività dell'utente per l'analisi delle prestazioni. Puoi visualizzare le statistiche individuali degli utenti:

    Metriche Disponibili

    • Transazioni Elaborate: Numero di vendite completate
    • Vendite Totali: Fatturato generato dall'utente
    • Transazione Media: Importo medio della vendita
    • Articoli Rimossi: Tracciamento degli articoli cancellati (per responsabilità)

    Accesso ai Report Utente

    1. Vai alla sezione Utenti
    2. Clicca sul nome dell'utente o sul pulsante Visualizza Report
    3. Seleziona l'intervallo di date per il report
    4. Rivedi le metriche delle prestazioni

    Considerazioni sulla Sicurezza

    La gestione utente è una funzione critica per la sicurezza. Segui queste linee guida per mantenere la sicurezza del sistema:

    Sicurezza degli Account

    • Privilegio Minimo: Concedi il ruolo più basso necessario per i doveri di ogni utente
    • Audit Regolari: Rivedi periodicamente gli account utente e rimuovi quelli non necessari
    • Password Forti: Applica pratiche di password forti
    • Azione Immediata: Disabilita o elimina account immediatamente quando i dipendenti se ne vanno

    Protezione degli Account Admin

    • Limita il numero di account admin
    • Usa password uniche e forti per ogni admin
    • Non condividere mai le credenziali admin
    • Effettua il logout quando lasci la postazione di lavoro incustodita
    Critico: Se un account admin è compromesso, cambia immediatamente la password e rivedi tutta l'attività recente per modifiche non autorizzate.
